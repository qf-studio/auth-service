## Problem

Add `golang-migrate/migrate` v4 as a dependency and create `cmd/migrate/main.go` — a CLI that accepts `up`, `down`, and `version` subcommands, reads `DATABASE_URL` (or builds DSN from the existing `PostgresConfig`), and runs embedded migrations from `migrations/`. This is a separate Go package from the SQL files and from `internal/storage/`. Include a basic test that validates the CLI builds and the migration source embeds correctly.
